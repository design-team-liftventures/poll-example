Polls Playground

A sandbox for pressure-testing the Polls feature — not a UI proposal, but a space to explore question formats, categories, and session lengths. Choosing a book, a category, and a session length generates a playground where the same takes run through all four poll formats, making it easy to compare how each one performs.
